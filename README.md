# TS_Study
TS Study

# TS의 특징
    1. 안정성을 위해 JS를 보완할 목적으로 나옴(런타임 에러 감소, 생산성 증가)
    2. 타입을 설정 할 수 있음, 타입을 생성 할수도 있음.
    3. 타입에 Readonly 를 적용해 불변값 으로 설정 가능 
    4. 특이한 타입으론 undefined, never(반환값이 절때 없을때, 에러를 발생시킬때), unknown(값을 모를때) 등이 존재함 
    5. TS의 타입체크를 활용하는게 핵심이다 (논리를 보호받기 위해)
    
# TS 다형성 
    정의 : 언어의 각 요소들(상수, 변수, 식, 오브젝트, 함수, 메소드) 등이 다양한 자료형에 속하는걸 허가되는걸 뜻함
    
    call signature : 함수의 인자 타입과 반환 타입
